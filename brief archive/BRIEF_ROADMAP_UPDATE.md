## What Changed

### v0.4 — MOD Mode Expansion (updated)
Added open source game modding as a new bullet under MOD Mode Expansion. Devforge will support modding games with publicly available source code — Doom, Quake, Diablo (devilution), Duke Nukem 3D, Wolfenstein 3D, SimCity (Micropolis), Prince of Persia. Added a line about curating open source tools and engines into quick links across all modes. This comes from a community resource catalog (Kavex/GameDev-Resources) that covers engines, asset tools, level editors, and more.
